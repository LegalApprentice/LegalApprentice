Legal Apprentice (LA) Suite Demo


This Azure deployment of LA-MPS ("Legal Apprentice - Marker, Pad, and Search") is intended to provide a functional demonstration to accompany the paper: Vern R. Walker and Stephen R. Strong, "Toward Implementation Science: A Case Study Using LA-MPS to Research Argument Elements at Scale," in Proceedings of the Sixth Workshop on Automated Semantic Analysis of Information in Legal Text (ASAIL 2023), June 23, 2023, Braga, Portugal.

This set of applications integrates to create LA-MPS:

Marker

Search

LA-Marker is a web application with a user interface for viewing and annotating individual legal documents. It is written using the open-source Angular web framework from Google. This Azure deployment is for demonstration purposes only — it will open and save files only locally, without communication to LA-Pad (see below).

LA-Pad is a web application normally integrated with LA-Marker, and has a user interface for grouping and annotating selections of sentences and paragraphs sent to it from LA-Marker or from LA-Search (see below). LA-Pad is omitted from this Azure deployment because it would require a login and authentication functionality.

LA-Search is a web application with a user interface for searching a large set of documents that have been semantically enriched. It is currently configured to use Elasticsearch, a search engine based on the Lucene library, but it could be configured to use any search engine. This Azure deployment is for demonstration purposes only — it will search only the stratified random sample dataset ("SRS Dataset") described in the paper cited above, with sentences curated only in the decision section headed "REASONS AND BASES FOR FINDINGS AND CONCLUSIONS". This Azure deployment is without communication to LA-Pad (see above).
